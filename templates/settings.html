ESP32 Network Sensor Code

Copy this Arduino sketch to your ESP32 to add wireless sensors to your wind tunnel controller. The code supports automatic discovery and remote configuration.

⚠ Setup Instructions:
  1. Install required libraries in Arduino IDE: WiFi, AsyncTCP, ESPAsyncWebServer
  2. Change WiFi credentials in the code (lines 11-12)
  3. Customize sensorID and sensorType (lines 15-16)
  4. Implement your sensor reading logic in readSensor() function (bottom of code)
  5. Upload to ESP32 and power on
  6. Click "Scan & Add Network Sensor" button above to discover and configure
